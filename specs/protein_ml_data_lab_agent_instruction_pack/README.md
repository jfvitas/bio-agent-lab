# Protein ML Data Lab — Agent Instruction Pack

This package contains the **full architectural instructions** for coding agents to build the Protein ML Data Lab system.

Purpose of the platform:
A **dataset engineering environment for structural biology ML** focused on:

• protein–protein binding prediction  
• protein–ligand binding prediction  
• mutation ΔΔG prediction  
• interaction pathway discovery  
• graph neural network datasets  

The system must allow users to:

1. Search structural biology databases
2. Build curated protein datasets
3. Download and clean structures
4. Extract structural and energetic features
5. Construct graph representations
6. Build unbiased ML training datasets
7. Export ML-ready data

All operations must be **accessible through a graphical interface**.

This pack defines:

- architecture
- GUI requirements
- workflow engine
- dataset engineering
- graph generation
- agent roles
