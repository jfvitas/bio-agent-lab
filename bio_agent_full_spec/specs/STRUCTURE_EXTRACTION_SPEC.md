
# Structure Extraction Specification

Primary structure source: RCSB PDB

Preferred file format: mmCIF

Download priority:

1 mmCIF (.cif)
2 BinaryCIF (.bcif)
3 PDB (.pdb)

Parser must extract:

- chain identities
- polymer entities
- ligand objects
- metal ions
- glycans
- cofactors
- crystallization additives
- biological assembly
